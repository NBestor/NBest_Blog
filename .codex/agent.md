## 一、工程化强制规范
### 1. 文件夹结构规范
- 小写字母、中横线分隔
- 功能内聚、单一职责
- 禁止中文、禁止特殊符号、禁止无意义命名

### 2. 文件命名规范
- 组件：`UpperCamelCase.jsx`
- 工具/接口/样式：`lower-snake-case.js` / `.css`
- 页面：`PageName.jsx`
- 配置：`config.js` / `.env`

### 3. 变量/函数命名
- 变量：`lowerCamelCase` 语义化
- 函数：`handleXxx` / `getXxx` / `setXxx` / `fetchXxx`
- 布尔值：`isXxx` / `hasXxx` / `canXxx`
- 常量：`UPPER_SNAKE_CASE`

### 4. 接口规范
- RESTful 风格
- 小写、复数、中横线
- 示例：`/api/articles`、`/api/quick-posts`

### 5. 权限规范
- 游客：`visitor`
- 用户：`user`
- 管理员：`admin`

---

## 二、数据库表命名规范
- 小写、下划线分隔
- 表名：复数形式
- 字段：小写、语义化
- 示例：`users`、`articles`、`quick_posts`

# 一、项目整体定位与设计原则
## 1. 项目定位
个人专属多功能博客系统，兼具：技术博客写作、社交关注、私密记事、待办管理、生日日历备忘、照片墙资源管理；支持游客浏览、注册用户互动、管理员全站管控，适配云服务器长期部署维护。

## 2. 设计原则
1. 代码极简轻量化，不引入冗余框架
2. UI 完全沿用你提供的博客 CSS 风格：简约白灰基调、圆角、轻微阴影、hover 过渡、代码块语法高亮
3. 前后端分离，解耦开发
4. 所有上传资源（头像/文章图片/照片）统一存服务器本地目录，Nginx 托管
5. 权限严格隔离：游客/普通用户/管理员、公开/仅好友/仅自己 三层权限
6. 所有提醒仅做**页面弹窗提醒**，无短信/邮箱推送
7. 快写、快记物理隔离，一个对外社交、一个私密自用

# 二、用户角色与权限体系（详细定义）
## 1. 三种角色
### （1）游客
- 可浏览：所有公开文章、公开快写、公开照片墙
- 不可操作：登录、注册、评论、点赞、收藏、发布任何内容、查看私密内容
- 不可进入个人中心、Todo、日历、快记等私密页面

### （2）普通注册用户
- 基础：登录/登出、修改个人资料、上传头像
- 内容：发布/编辑/删除自己文章、草稿管理、发布快写、快记
- 互动：评论、点赞、收藏、关注其他用户、双向关注成好友
- 工具：使用Todo待办、日历备忘录、个人照片墙管理
- 权限：只能管理自己的内容，无法干预他人内容

### （3）超级管理员（仅你一人）
- 拥有普通用户全部权限
- 额外权限：删除任意用户文章、评论、快写
- 查看所有用户私密内容（可选，后期可加）
- 管理全站照片资源

## 2. 全局内容可见范围规则
1. 文章：公开 / 仅好友可见 / 仅自己可见
2. 快写：全局可在个人设置中**设置默认可见范围**，单条也可独立修改
3. 快记：强制仅自己可见，无对外权限
4. 照片：单张可设置 公开 / 仅自己可见
5. Todo、日历备忘录：强制仅自己可见

# 三、全功能模块详细需求设计
## 模块1：用户账号系统
### 1. 注册
- 字段：用户名、密码、确认密码、昵称
- 无手机/邮箱验证码，简化私人使用
- 密码后端哈希加密存储，不存明文
- 注册成功自动创建个人默认头像目录、照片墙目录

### 2. 登录/登出
- 账号密码登录，颁发JWT令牌
- 前端LocalStorage存储Token，页面刷新保持登录态
- Token有效期7天，自动续期
- 登出清空Token，跳转首页

### 3. 头像上传
- 支持本地图片上传
- 后端压缩裁剪，统一尺寸
- 存储到服务器固定目录，Nginx提供访问链接
- 个人中心可随时更换、重置

### 4. 个人中心 & 设置
- 资料修改：昵称、个性签名、头像
- 密码修改
- **快写默认可见范围设置**（全局生效）
- 我的内容：我的文章、我的草稿、我的收藏、我的评论、我的快写、我的快记、我的照片
- 社交：我的关注、我的粉丝、我的好友列表

### 5. 用户关注/好友机制
- 用户可关注其他用户
- 单向：仅关注，不是好友
- 双向互相关注：自动成为好友
- 好友可查看彼此「仅好友可见」的文章
- 可取消关注、移除好友

---

## 模块2：Markdown 博客文章系统
### 1. Markdown 编辑器
- 左右双栏布局：左编辑、右实时预览
- 支持标准Markdown：标题、加粗、列表、引用、表格、代码块、换行
- 编辑器工具栏：快捷插入标题、代码块、引用、上传图片
- **云端草稿功能**：
  - 未发布文章自动保存为草稿
  - 可随时进入草稿继续编辑
  - 可删除草稿、草稿直接发布为正式文章

### 2. 图片上传（文章插图）
- 编辑内直接本地上传图片
- 后端接收保存到服务器图片目录
- 自动返回图片URL插入Markdown内容
- 所有上传图片**自动同步归入个人照片墙**

### 3. 文章分类+标签
- 一级**固定分类**（后端可后台新增分类）
- 二级**自定义标签**（用户发布时自由输入创建）
- 文章列表支持按分类筛选、标签筛选

### 4. 文章发布与权限
- 发布选项：公开 / 仅好友可见 / 仅自己可见
- 可填写文章简介摘要
- 可随时编辑、删除自己文章
- 管理员可删除任意文章

### 5. 文章浏览
- 游客可看公开文章
- 登录用户可看公开+好友可见
- 仅自己可见仅限作者本人查看
- **不记录阅读量**

---

## 模块3：评论 / 点赞 / 收藏 互动系统
### 1. 评论
- 仅登录用户可评论，游客禁止
- 支持楼中楼回复（回复指定单条评论）
- 纯文本评论，无需富文本
- 自己可删除自己评论，管理员可删除全部
- 无需审核，发布立即展示

### 2. 点赞
- 支持：文章点赞、评论点赞、快写点赞
- 同一用户对同一内容只能点一次，可取消点赞
- **点赞数量所有人可见（游客+登录用户都能看）**

### 3. 收藏
- 登录用户可收藏任意公开文章
- 个人中心可查看所有收藏文章
- 可取消收藏

---

## 模块4：照片墙系统（独立专属模块）
### 1. 图片来源
- 用户头像
- Markdown文章上传插图
- 后续手动上传的个人照片
- 全部自动归集到个人照片墙

### 2. 功能细节
- 网格/瀑布流布局，沿用全站简约风格
- 单张照片可设置：公开 / 仅自己可见
- 可预览大图、复制图片链接
- 可删除自己上传的照片
- 管理员可管理全站照片

### 3. 权限
- 游客/其他用户只能看公开照片
- 私密照片仅自己可见

---

## 模块5：快写 & 快记 双模块完全分离
### 1. 快写（To Others 对外动态）
- 纯文本输入，**不配图**
- 发布后进入全站公开动态流
- 单条可单独设置可见范围
- 遵循个人中心默认可见范围配置
- 支持点赞、评论
- 仅自己可删除，管理员可删除任意

### 2. 快记（To Self 私密记事本）
- 纯文本输入，不配图
- **强制仅自己可见**，永远不对外展示
- 无点赞、无评论、无对外流量
- 用于随手碎碎念、临时记录、私密想法
- 可无限新增、编辑、删除

---

## 模块6：TodoList 待办清单
1. 支持新增、编辑、删除、标记完成/未完成
2. 固定分类：工作、生活、学习
3. 每条待办可设置**截止日期**
4. **页面弹窗提醒**：进入网站自动弹窗提醒已到期/今日即将到期待办
5. 所有待办强制仅自己可见

---

## 模块7：日历备忘录（生日/纪念日）
1. 月历视图展示日历
2. 可添加事件：标题、日期、备注、事件类型（生日/纪念日/其他）
3. 支持**每年重复**（生日自动跨年显示）
4. **页面弹窗提醒**：进入网站弹窗展示未来7天即将到来的事件
5. 不做农历，仅公历记录
6. 全部内容仅自己可见
